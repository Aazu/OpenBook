OpenBooks — Local Static Demo (No Node / No Python required)

How to run locally on Windows:
1) Extract this ZIP.
2) Double-click index.html to open in Chrome/Edge.
   - (Optional recommended) Use VS Code Live Server for best behavior.

Pages:
- index.html                -> Consumer Portal (browse/search/view/rate/comment)
- creator_console.html      -> Creator Console (upload + metadata)
- admin_console.html        -> Admin Console (manage creators + moderate posts)
- settings_profile.html     -> Settings/Profile (switch user + change role)
- tech_status.html          -> Tech Status (mock cloud health + storage usage)

Dummy Database:
- Stored in your browser (localStorage).
- Click “Reset demo DB” to reseed demo data.

Tip:
- Uploads are compressed automatically to reduce storage and prevent freezing.
